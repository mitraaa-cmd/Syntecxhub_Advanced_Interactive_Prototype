# 🍔 FoodieGo – Advanced Interactive Prototype

An interactive food delivery app prototype built using **HTML, CSS, and JavaScript**. This project demonstrates modern UI/UX principles with smooth micro-interactions, loading animations, transitions, and multiple connected screens to simulate a real mobile application.

## 📱 Features

- ✨ Animated Splash Screen
- 🔐 Interactive Login Screen
- 🔍 Expandable Search Bar
- ❤️ Favorite (Heart) Animation
- 🍔 Restaurant Listing Cards
- 📋 Restaurant Details Page
- 🍟 Food Details Bottom Sheet
- 🔍 Image Zoom Animation
- ➕ Quantity Selector
- 🛒 Add to Cart Animation
- 💳 Checkout Loading Animation
- 🎉 Order Success Screen with Confetti
- 📱 Mobile Phone Mockup Layout
- 🎨 Modern UI Design
- ⚡ Smooth CSS Transitions & Micro-interactions
